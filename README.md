# AwesomeSites

> 很多精彩的网站随着时间推移很有可能会消失在互联网上，所以我建立了这个仓库，将这些值得学习的网站收藏并备份下来。本仓库只是一个目录。
## 使用方法

进入网站域名文件夹后开启`live-server`或其他的本地http服务后即可离线运行。
